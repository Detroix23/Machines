# I2P.
Basics.

## CLI. 
Do **not** run as **sudo** !

On-Demand:
```bash
i2prouter start
i2prouter stop
```

Without Java service wrapper:
```
i2prouter-nowrapper start
i2prouter-nowrapper stop
```

System service, auto-startup:
```
sudo dpkg-reconfigure i2p
```

## Server.
Port:

http://127.0.0.1:7657
